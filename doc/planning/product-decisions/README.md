# Product Decisions

Product-level decisions made by `@pm` during change delivery.

## Purpose

When agents surface product decisions during change delivery, `@pm` documents them here with:

- Context and options considered
- Decision drivers and rationale
- Consequences and follow-up actions

## Naming Convention

Files follow: `YYYY-MM-DD-<short-kebab-slug>.md`

## Reference

See the `@pm` agent's `<product_decisions>` workflow for how these decisions are captured.
